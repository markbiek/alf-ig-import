This plugin will automatically post content to Bluesky.

== Description ==
This plugin will automatically post content to Bluesky when a post is published. Users can select which categories to post to Bluesky.

If the post is less than 300 characters, the entire post will be posted. If the post is longer than 300 characters, the post excerpt will be posted along with a link to the full post.

The post title is always ignored.

== Installation ==
1. Upload "alf-bsky-poster.zip" via the WordPress admin or install from the WordPress repository.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Go to the "Settings" menu and set the Bluesky API credentials. We recommend using a Bluesky application password specifically for this plugin and not your main Bluesky account password.
